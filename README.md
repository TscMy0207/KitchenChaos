## **游戏联机**
游戏连接采用官方的Unity Multiplayer
游戏中的网络变量采用的是NetworkVariable 网络同步用的RPC(降低延迟)

### 联机界面
![联机界面](/DemoImg/联机界面.png)
![联机界面](/DemoImg/角色选择界面.png)
可以设置最多加入人数 人数超过最大人数 客户端会提示失败
通过自定义网络变量 在通过服务器函数动态修改 客户端是否准备 客户端玩家的颜色
